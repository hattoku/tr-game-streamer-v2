# コード規約

## ファイル冒頭のコメント

各ファイルの冒頭には日本語のコメントで、そのファイルが何をするものかを記述する。

```typescript
/**
 * 信頼度スコアを計算するユーティリティ
 * 計算式: W = (D + H + G) × C
 * 詳細は `document/specification/common/信頼度スコアリングシステム仕様書` を参照
 */
```

対応する仕様書がある場合は、仕様書へのパスも記載する。

## 仕様書との対応を明示する

仕様書に定義された重要なロジック（権限判定・スコア計算・審査フロー等）を実装するコードには、参照元の仕様書と章番号をコメントで記載する。

```typescript
// 審査者アサインルール: 管理 審査ワークフロー仕様書 第4章
if (!gameTitle) {
  // ゲームタイトルが紐づいていない場合はSTEP 1をスキップしてSTEP 2へ
  return assignToStep2();
}
```
