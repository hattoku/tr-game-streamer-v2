## 1. 概要

本仕様書は、ログイン済みユーザーが自分のお問い合わせ履歴とスレッドを確認・返信できる、お問い合わせ履歴ページ（`/contact/history`）の仕様を定義する。

---

## 2. アクセス条件・導線

| 項目 | 内容 |
|------|------|
| URL | `/contact/history` |
| アクセス条件 | ログイン済みユーザーのみ |

### 導線

| 導線元 | 内容 |
|--------|------|
| お問い合わせ送信完了画面 | 送信完了後に履歴ページへのリンクを表示 |
| 設定ページ（`/settings`） | 履歴確認用のリンクを設置（検討中） |
| 通知 | 管理者からの返信通知から直接該当スレッドへ遷移 |

---

## 3. 履歴一覧画面

ログイン済みユーザー自身のお問い合わせ一覧を表示する。

### 表示項目

| 列 | 内容 |
|----|------|
| 受付日時 | お問い合わせが送信された日時 |
| 種別 | お問い合わせ種別 |
| ステータス | 未対応・対応中・保留中・対応済みのバッジ表示 |
| 未読インジケーター | 管理者（オーナー・運営者）からの未読メッセージがある場合に表示 |

- 行をクリックするとスレッド詳細画面へ遷移する。
- デフォルトのソート順は受付日時の降順（新しい順）。

## 4. スレッド詳細画面

### 画面構成

```
┌─────────────────────────────────────────────────┐
│ ヘッダー                                         │
├─────────────────────────────────────────────────┤
│                                                 │
│ ← お問い合わせ一覧へ戻る                          │
│                                                 │
│ 不具合・バグ報告                   [ 未対応 ]     │
│ 受付日時: 2026-03-08 14:32                       │
│                                                 │
│ ── お問い合わせ内容 ──────────────────────────── │
│ 動画プレーヤーで○○をすると画面が固まります。      │
│                                                 │
│ 関連URL                                         │
│ https://gamecome.jp/player/xxxxx                │
│                                                 │
│ ── やり取り ──────────────────────────────────── │
│                                                 │
│  あなた  2026-03-08 14:32                        │
│  動画プレーヤーで○○をすると…                    │
│                                                 │
│            運営  2026-03-08 15:00               │
│            ご報告ありがとうございます。           │
│                                                 │
│ [ メッセージを入力                  ]            │
│                              [ 送信する ]        │
│                                                 │
│ ※ スレッドがクローズされている場合は入力不可       │
│ [ 再オープンする ]                               │
│                                                 │
└─────────────────────────────────────────────────┘
```

### スレッドの状態による表示切り替え

| 状態 | 入力エリア | 送信ボタン | 再オープンボタン |
|------|:---:|:---:|:---:|
| オープン中 | 表示 | 表示 | 非表示 |
| クローズ済み | 非表示 | 非表示 | 表示 |

---

## 5. 再オープン機能

スレッドが管理者によってクローズされた後、ユーザーが追加で連絡が必要な場合に再開できる機能。

### 再オープンフロー

```
「再オープンする」ボタンをクリック
    ↓
スレッドを再オープン
ステータスを「未対応」に自動変更
入力エリアおよび送信ボタンを再表示
```

- 再オープンが行われると、管理者側へ通知が送信される。
