## 1. 概要

### 1.1 機能概要

まとめ機能は、ログイン済みユーザーが複数の再生リストを「まとめ」として組み、説明文とともに友人にシェアできる機能である。

各再生リストには、ユーザーが既に投稿済みのレビューコメントが自動的に紐づいて表示される。「好きな実況を友人に勧めたい」という自然な動機から、レビュー投稿への動機づけを生む設計になっている。

### 1.2 機能の位置づけ

初期フェーズから実装するユーザー動機づけ機能である。LINEなどでの友人シェアをメインユースケースとして設計する。

## 2. アクセス条件・権限

### 2.1 作成・編集・削除

|条件        |内容          |
|----------|------------|
|ログイン済みユーザー|✅ 作成・編集・削除可能|
|未ログインユーザー |❌ 作成・編集・削除不可|

### 2.2 閲覧

|まとめの公開設定|未ログインユーザー       |ログイン済みユーザー（他人）  |作成者本人 |
|--------|----------------|----------------|------|
|公開      |✅ 閲覧可能          |✅ 閲覧可能          |✅ 閲覧可能|
|限定公開    |✅ URLを知っていれば閲覧可能|✅ URLを知っていれば閲覧可能|✅ 閲覧可能|
|非公開     |❌ 閲覧不可          |❌ 閲覧不可          |✅ 閲覧可能|

### 2.3 保存

|条件        |内容                |
|----------|------------------|
|ログイン済みユーザー|✅ 公開・限定公開のまとめを保存可能|
|未ログインユーザー |❌ 保存不可            |

## 3. まとめの構造

### 3.1 まとめの構成要素

```
【まとめ】タイトル（必須）
説明文（任意）

  ├─ 再生リストA
  │   └─ 自分のレビューコメント（投稿済みの場合に表示）
  │
  ├─ 再生リストB
  │   └─ 自分のレビューコメント（投稿済みの場合に表示）
  │
  └─ 再生リストC
      └─ （レビュー未投稿の場合はコメント非表示）
```

### 3.2 各項目の仕様

|項目      |必須/任意|仕様                                                |
|--------|-----|--------------------------------------------------|
|タイトル    |必須   |最大50文字                                            |
|説明文     |任意   |まとめ全体に対して1件。最大500文字                               |
|再生リスト   |必須   |1件以上10件以下                                         |
|レビューコメント|自動表示 |各再生リストに対してユーザーが投稿済みのレビューコメントを自動的に紐づけて表示。未投稿の場合は非表示|

### 3.3 制限

|項目             |上限 |
|---------------|---|
|1まとめあたりの再生リスト数 |10件|
|1ユーザーあたりのまとめ作成数|10件|

### 3.4 公開設定

|設定  |挙動                              |
|----|--------------------------------|
|公開  |ゲムコメ上の検索・一覧にヒットする。URLでもアクセス可能   |
|限定公開|検索・一覧にはヒットしない。URLを知っている人のみアクセス可能|
|非公開 |作成者本人のみ閲覧可能                     |

デフォルトは**非公開**。

### 3.5 ネタバレコメントの表示オプション

まとめに含まれる再生リストのレビューコメントにネタバレフラグが付いている場合、作成者はまとめ単位でネタバレコメントの表示・非表示を選択できる。

|設定          |挙動                              |
|------------|--------------------------------|
|表示する        |ネタバレコメントをそのまま表示する               |
|表示しない（デフォルト）|ネタバレコメントを「ネタバレを含みます」として折りたたみ表示する|

## 4. まとめ作成・編集機能

### 4.1 作成導線

まとめの作成は以下の3箇所から行える。

|導線                  |UI要素          |
|--------------------|--------------|
|プロフィールページのまとめセクション  |「新しいまとめを作る」ボタン|
|再生リスト詳細ページの視聴記録セクション|「まとめに追加する」ボタン |
|マイリストページ            |「まとめに追加する」ボタン |

再生リスト詳細ページおよびマイリストページの「まとめに追加する」ボタンからは、既存まとめへの追加と新規まとめの作成の両方に対応する。

### 4.2 作成・編集フォームの仕様

```
┌─────────────────────────────────────────────────┐
│ まとめを作成する                                  │
│                                                 │
│ タイトル *                                       │
│ ┌─────────────────────────────────────────────┐ │
│ │                                             │ │
│ └─────────────────────────────────────────────┘ │
│                                                 │
│ 説明文                                           │
│ ┌─────────────────────────────────────────────┐ │
│ │                                             │ │
│ └─────────────────────────────────────────────┘ │
│                                                 │
│ 公開設定                                         │
│ ○ 非公開  ○ 限定公開  ○ 公開                    │
│                                                 │
│ ネタバレコメント                                  │
│ ○ 表示しない  ○ 表示する                         │
│                                                 │
│ 再生リスト（最大10件）                            │
│ ┌─────────────────────────────────────────────┐ │
│ │ ≡  サムネイル  再生リストタイトル  ↑ ↓  [削除] │ │
│ │ ≡  サムネイル  再生リストタイトル  ↑ ↓  [削除] │ │
│ └─────────────────────────────────────────────┘ │
│ [＋ 再生リストを追加する]                         │
│                                                 │
│              [キャンセル]  [保存する]             │
└─────────────────────────────────────────────────┘
```

### 4.3 再生リストの並び替え

まとめ内の再生リストの順番は、以下の2つの操作で変更できる。

|操作       |詳細                                      |
|---------|----------------------------------------|
|ドラッグ＆ドロップ|各行左端の「≡」アイコンをドラッグして並び替える。モバイルでは長押し後にドラッグ|
|↑↓ボタン    |各行に設置された上下移動ボタンで1つずつ移動する                |

### 4.4 バリデーション

|項目   |バリデーション内容                  |
|-----|---------------------------|
|タイトル |1文字以上50文字以内。空欄の場合は保存不可     |
|説明文  |500文字以内。超過時はカウンターを赤色表示し保存不可|
|再生リスト|1件以上。0件の場合は保存不可            |

### 4.5 まとめの並び替え

プロフィールページのまとめセクションおよびまとめ管理画面において、まとめ自体の表示順を並び替えられる。

|操作       |詳細                        |
|---------|--------------------------|
|ドラッグ＆ドロップ|まとめカードをドラッグして並び替える        |
|↑↓ボタン    |各カードに設置された上下移動ボタンで1つずつ移動する|

デフォルトの並び順は作成日時の降順（新しい順）。

### 4.6 まとめの削除

まとめは作成者本人が削除できる。削除すると復元はできない。

## 5. まとめ詳細ページ

### 5.1 ページレイアウト

```
┌─────────────────────────────────────────────────┐
│ ヘッダー                                         │
├─────────────────────────────────────────────────┤
│                                                 │
│  まとめタイトル                                   │
│  作成者名  作成日                                 │
│                                                 │
│  説明文（任意）                                   │
│                                                 │
│  [まとめを保存する ♡]   [編集する]（作成者のみ）  │
│                                                 │
│ ─────────────────────────────────────────────── │
│                                                 │
│  ┌─────────────────────────────────────────┐   │
│  │ 再生リストカード 1                         │   │
│  └─────────────────────────────────────────┘   │
│  ┌─────────────────────────────────────────┐   │
│  │ 再生リストカード 2                         │   │
│  └─────────────────────────────────────────┘   │
│   ...                                          │
│                                                 │
└─────────────────────────────────────────────────┘
```

### 5.2 ヘッダー情報エリアの表示要素

|要素     |詳細                                |
|-------|----------------------------------|
|まとめタイトル|作成者が設定したタイトル                      |
|作成者名   |クリックで作成者のプロフィールページへ遷移             |
|作成日    |「YYYY年M月D日」の形式で表示                 |
|説明文    |任意入力のため、未入力の場合は非表示                |
|保存ボタン  |ログイン済みユーザーに表示。保存済みの場合はアイコンを塗りつぶし表示|
|編集ボタン  |作成者本人のみ表示                         |

### 5.3 再生リストカードの表示要素

```
┌──────────────────────────────────────────────────┐
│ ┌──────────┐  再生リストタイトル                   │
│ │          │  ゲームタイトル名                     │
│ │サムネイル │                                     │
│ │          │  ★ 4.2  [視聴ステータスバッジ]        │
│ └──────────┘                                     │
│                                                  │
│  レビューコメント                                  │
│  （ネタバレ設定に応じて表示・折りたたみ）           │
└──────────────────────────────────────────────────┘
```

|要素         |詳細                                                                           |
|-----------|-----------------------------------------------------------------------------|
|再生リストのサムネイル|クリックで再生リスト詳細ページへ遷移                                                           |
|再生リストタイトル  |クリックで再生リスト詳細ページへ遷移。最大2行、超過時は省略記号（…）                                          |
|ゲームタイトル名   |クリックでゲームタイトル詳細ページへ遷移                                                         |
|再生リストスコア    |★アイコン＋小数点第1位（例：★ 4.2）。レビュー0件の場合は「評価なし」                                       |
|視聴ステータスバッジ |作成者の視聴ステータス。設定されている場合のみ表示                                                    |
|レビューコメント   |作成者が当該再生リストに投稿済みのレビューコメント。未投稿の場合は非表示。ネタバレフラグがある場合はまとめのネタバレ設定に従い表示・折りたたみを切り替える|

## 6. まとめ専用検索・一覧ページ

### 6.1 ページ構成

```
┌─────────────────────────────────────────────────┐
│ ヘッダー                                         │
├─────────────────────────────────────────────────┤
│ まとめを探す                                      │
│                                                 │
│ ┌─────────────────────────────────────────────┐ │
│ │ 🔍 キーワードで検索                           │ │
│ └─────────────────────────────────────────────┘ │
│                                                 │
│ 並び順: [新着順 ▼]                               │
│         └ 新着順                                │
│         └ 保存数順                              │
│                                                 │
│  ┌─────────────────────────────────────────┐   │
│  │ まとめカード                               │   │
│  └─────────────────────────────────────────┘   │
│   ...                                          │
└─────────────────────────────────────────────────┘
```

### 6.2 検索・ソートの仕様

|機能      |仕様                       |
|--------|-------------------------|
|キーワード検索 |まとめのタイトル・説明文を対象にキーワード検索する|
|ソート：新着順 |作成日時の降順（デフォルト）           |
|ソート：保存数順|保存された回数の降順               |

検索・一覧に表示されるのは**公開設定が「公開」のまとめのみ**。

### 6.3 まとめカードの表示要素

```
┌──────────────────────────────────────────────────┐
│ まとめタイトル                                     │
│ 作成者名  作成日                                   │
│                                                  │
│ 説明文の冒頭（最大100文字程度、超過時は省略）        │
│                                                  │
│ [サムネイル] [サムネイル] [サムネイル]  全X件        │
│                                                  │
│ ♡ 保存数                                         │
└──────────────────────────────────────────────────┘
```

|要素         |詳細                                                |
|-----------|--------------------------------------------------|
|まとめタイトル    |クリックでまとめ詳細ページへ遷移                                  |
|作成者名       |クリックで作成者のプロフィールページへ遷移                             |
|作成日        |「YYYY年M月D日」の形式で表示                                 |
|説明文の冒頭     |任意項目のため、未入力の場合は非表示                                |
|再生リストのサムネイル|まとめに含まれる再生リストのサムネイルを最大3件並べて表示。件数が3件を超える場合は「全X件」を付記|
|保存数        |♡アイコン＋保存された回数                                     |

### 6.4 ページネーション

20件/ページ、ページ番号リンク形式。

## 7. 発見導線

公開まとめは以下の5箇所から発見・アクセスできる。

|場所           |表示内容                                     |
|-------------|-----------------------------------------|
|まとめ専用検索・一覧ページ|キーワード検索・ソートにより公開まとめを一覧表示                 |
|トップページ       |新着・保存数の多い公開まとめを一部表示（詳細仕様はトップページ仕様書に定義）   |
|プロフィールページ    |そのユーザーが作成した公開まとめをじゃばら展開で表示（第9章参照）        |
|再生リスト詳細ページ   |その再生リストを含む公開まとめを表示（詳細仕様は再生リスト詳細ページ仕様書に定義）|
|マイリストページ     |まとめへの追加導線を設置（詳細仕様はマイリスト機能仕様書に定義）         |

## 8. 保存機能

### 8.1 概要

ログイン済みユーザーが、他のユーザーが作成した公開・限定公開のまとめを保存できる機能。UIには「まとめを保存する」と明示する。

### 8.2 保存の操作

|操作           |挙動                            |
|-------------|------------------------------|
|保存ボタンを押す     |まとめを保存する。ボタンアイコンを塗りつぶし表示に切り替える|
|保存済みのボタンを再度押す|保存を解除する。ボタンアイコンを通常表示に戻す       |

自分が作成したまとめには保存ボタンを表示しない。

### 8.3 保存したまとめの確認場所

保存したまとめは以下の2箇所から確認できる。

|場所                      |詳細                               |
|------------------------|---------------------------------|
|ヘッダーの「保存したまとめ」リンク       |保存したまとめの専用一覧ページへ遷移。ログイン済みユーザーのみ表示|
|プロフィールページの「保存したまとめ」セクション|じゃばら展開で一覧表示（第9章参照）               |

### 8.4 保存したまとめ一覧ページの仕様

|項目      |仕様                            |
|--------|------------------------------|
|表示対象    |自分が保存したまとめ（公開・限定公開問わず）        |
|並び順     |保存日時の降順（新しく保存した順）             |
|カード形式   |まとめ専用検索・一覧ページのカード仕様（第6.3節）に準ずる|
|ページネーション|20件/ページ、ページ番号リンク形式            |

## 9. プロフィールページとの連携

### 9.1 まとめセクション

プロフィールページに「まとめ」セクションを追加する。表示されるのは**公開設定が「公開」のまとめのみ**。じゃばら展開の共通仕様（プロフィールページ仕様書 第3.2節）に準ずる。

```
┌─────────────────────────────────────────────────┐
│ まとめ（3件）                          [作る ＋]  │
│                                                 │
│  ┌─────────────────────────────────────────┐   │
│  │ まとめカード 1                             │   │
│  └─────────────────────────────────────────┘   │
│  ┌─────────────────────────────────────────┐   │
│  │ まとめカード 2                             │   │
│  └─────────────────────────────────────────┘   │
│  ┌─────────────────────────────────────────┐   │
│  │ まとめカード 3                             │   │
│  └─────────────────────────────────────────┘   │
│                                                 │
│           [ すべて見る（3件）▼ ]                │
└─────────────────────────────────────────────────┘
```

|要素       |詳細                                       |
|---------|-----------------------------------------|
|「作る ＋」ボタン|自分のプロフィールページを閲覧中の場合のみ表示。クリックで新規まとめ作成フォームへ|
|カード      |まとめ専用検索・一覧ページのカード仕様（第6.3節）に準ずる           |
|並び順      |手動並び替えの設定順。未設定の場合は作成日時の降順                |

### 9.2 保存したまとめセクション

プロフィールページに「保存したまとめ」セクションを追加する。本人のページでのみ表示する（他人のプロフィールページには表示しない）。じゃばら展開の共通仕様（プロフィールページ仕様書 第3.2節）に準ずる。

|項目   |仕様                            |
|-----|------------------------------|
|表示対象 |自分が保存したまとめ（公開・限定公開問わず）        |
|並び順  |保存日時の降順                       |
|カード形式|まとめ専用検索・一覧ページのカード仕様（第6.3節）に準ずる|
