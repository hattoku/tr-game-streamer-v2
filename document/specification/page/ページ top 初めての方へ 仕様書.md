## 1. 概要

「初めての方へ」ページは、プレミテを初めて知ったユーザーに対してサービスの概要と主要機能を説明し、アカウント登録へ誘導するページである。

プレミテは「ゲーム実況動画の再生リスト視聴記録・新着通知サービス」という、既存サービスにない独自のコンセプトを持つ。アカウント不要でもコンテンツを閲覧できる設計のため、ユーザーが「このサービスで何ができるのか」を能動的に知ろうとするタイミングで参照する場として機能する。

## 3. 流入導線

### 3.1 TOPページ ヒーローセクションからの導線

TOPページのヒーローセクション（未ログイン時のみ表示）に、テキストリンクを設置する。

| 項目 | 内容 |
|------|------|
| 表示条件 | 未ログイン時のみ表示（ヒーローセクションと同じ表示条件に準ずる） |
| リンク文言 | 「初めての方へ →」 |
| 配置位置 | 検索バーの下部 |
| リンク形式 | テキストリンク（目立たせすぎない小さめのスタイル） |

### 3.2 フッターからの導線

全ページ共通のフッターにリンクを常設する。

| 項目 | 内容 |
|------|------|
| 表示条件 | ログイン・未ログイン問わず常時表示 |
| リンク文言 | 「初めての方へ」 |
| 配置位置 | フッター内のサービス案内リンク群の中に配置 |

## 5. 各セクションの仕様

### 5.1 サービス紹介セクション

プレミテとは何かを簡潔に説明する冒頭セクション。

#### 表示要素

| 要素 | 仮文言 |
|------|--------|
| 見出し | 「プレミテへようこそ」 |
| サービス説明文 | 「プレミテは、ゲーム実況動画の再生リストを追いかけるためのサービスです。気に入った再生リストをマイリストに登録して視聴記録を管理したり、新着動画の通知を受け取ったり、ほかのユーザーのレビューを参考にしたりできます。アカウントなしでも再生リストの閲覧・視聴ができます。」 |

---

### 5.2 機能紹介セクション

プレミテの主要機能を5つ、順番に紹介する。各機能は「機能タイトル」「説明文」「スクリーンショットまたはイメージ画像」で構成する。

#### 共通レイアウト（PC）

各機能ブロックは、画像と説明文を左右2カラムで配置する。奇数番目（1・3・5）は画像左・テキスト右、偶数番目（2・4）は画像右・テキスト左の交互配置とする。

#### 機能① 再生リストを探す

| 項目 | 内容 |
|------|------|
| 機能タイトル | 「好きなゲームの実況シリーズを見つけよう」 |
| 説明文（仮） | 「ゲームタイトル・チャンネル名・キーワードからお気に入りの実況シリーズを探せます。ジャンルやタグで絞り込んで、あなた好みの再生リストに出会いましょう。」 |

#### 機能② マイリスト

| 項目 | 内容 |
|------|------|
| 機能タイトル | 「複数の再生リストをまとめて管理」 |
| 説明文（仮） | 「気になる再生リストをマイリストに追加して、視聴ステータス（視聴中・見たい・完走など）を管理できます。複数の再生リストを並行して追いかけるときも、マイリストがあれば迷いません。」 |

#### 機能③ 新着通知

| 項目 | 内容 |
|------|------|
| 機能タイトル | 「新しい動画が来たらすぐわかる」 |
| 説明文（仮） | 「完走した再生リストに新しい動画が追加されると、マイリスト上で通知します。続編や追加エピソードを見逃さずに追いかけられます。」 |

#### 機能④ 視聴記録

| 項目 | 内容 |
|------|------|
| 機能タイトル | 「どこまで見たか、ちゃんと覚えてる」 |
| 説明文（仮） | 「動画を開くと自動で視聴履歴が記録されます。アカウントを作成すれば端末をまたいで記録が引き継がれるので、スマホで続きを見ることも簡単です。」 |

#### 機能⑤ レビュー・スコア

| 項目 | 内容 |
|------|------|
| 機能タイトル | 「ちゃんと見た人の、信頼できる評価」 |
| 説明文（仮） | 「各再生リストにはユーザーのレビューとスコアが表示されます。プレミテのスコアは視聴深度や投稿の質を加味した加重平均なので、実際に見た人の声が正直に反映されます。気になる再生リストを選ぶときの参考にどうぞ。」 |

---

### 5.3 CTAセクション

機能紹介セクションの直下に配置する。アカウント登録への最終的な誘導を行う。

#### 表示要素

| 要素 | 内容 |
|------|------|
| 見出し（仮） | 「さっそく使ってみましょう」 |
| 補足文（仮） | 「アカウント登録は無料です。Google・Xアカウントで簡単に始められます。」 |
| CTAボタン | 「無料でアカウントを作成する」。クリックでアカウント作成フォーム（`/signup`）へ遷移 |
| サブリンク | 「すでにアカウントをお持ちの方はこちら」。クリックでログインページ（`/login`）へ遷移 |

#### ボタン仕様

| 項目 | 内容 |
|------|------|
| ボタン色 | ブランドカラー（#E53935）またはブラック（#212121）。デザイン実装時に決定する |
| 配置 | 中央揃え |

## 6. スマートフォン対応

### 6.1 機能紹介セクション

PCでの左右2カラム交互配置を、スマートフォンでは「画像上・テキスト下」の1カラム縦積みに変更する。交互配置はせず、すべて同じ順序で統一する。
