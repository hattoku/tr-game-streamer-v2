## 1. 概要

プライバシーポリシーページは、ゲムコメにおける個人情報の取り扱いを掲載するページである。

プライバシーポリシーの本文は後日整備する。本仕様書では、ページの存在とアクセス導線を定義することを目的とする。

本文には以下の事項を含めることとする（詳細文面は後日整備）。

| 記載事項 | 概要 |
|---------|------|
| 収集する個人情報 | アカウント登録時に取得するメールアドレス |
| 利用目的 | アカウント認証、サービス運営に関する連絡 |
| Google Analytics 4（GA4）の利用 | アクセス解析を目的としてGA4を使用しており、Cookieおよびブラウザ識別子を通じた行動データを収集することを明示する |
| Cookieの利用 | Firebase AuthenticationによるセッションCookieを使用することを明示する |
| 第三者提供 | 原則として第三者への提供は行わない旨 |
| 問い合わせ窓口 | 個人情報に関する問い合わせ先 |

## 3. 流入導線

### 3.1 フッターからの導線

全ページ共通のフッターにリンクを常設する。

| 項目 | 内容 |
|------|------|
| 表示条件 | ログイン・未ログイン問わず常時表示 |
| リンク文言 | 「プライバシーポリシー」 |
| 配置位置 | フッター内のリンク群に配置 |

### 3.2 アカウント登録フローからの導線

アカウント作成フォームに、プライバシーポリシーへの同意文を設置する。

| 項目 | 内容 |
|------|------|
| 配置位置 | 「アカウントを作成する」ボタンの直上 |
| 表示文言 | 「アカウントを作成することで、[利用規約]および[プライバシーポリシー]に同意したものとみなします。」 |
| リンク | 「プライバシーポリシー」部分をリンクとし、クリックでプライバシーポリシーページへ遷移する |
| 遷移方式 | 別タブで開く（`target="_blank"`） |

> ソーシャルログイン（Google・X）経由でのアカウント作成にも同様の同意文を表示する。各プロバイダのソーシャルログインボタンの直上に配置する。
