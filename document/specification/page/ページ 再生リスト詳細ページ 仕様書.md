## 概要

### 目的

再生リスト（ゲーム実況シリーズ）の詳細情報を、ユーザーが効率的に把握できるページレイアウト・UI構成を提供する。動画プレーヤーを中心に、再生リストの基本情報、配信者情報、ゲーム情報を有機的に配置し、ユーザーの導線を最適化する。

### スコープ

- ページ全体のレイアウト・2カラム構成
- ページ内各セクションの配置順序と役割
- 各セクションの表示要素と表示形式
- ページ上のセクション間の相互作用（シアターモード時を含む）
- レスポンシブ対応（PC / モバイル）

### 非スコープ

- YouTube埋め込みプレーヤーの技術実装（別仕様書）
- 動画リストの詳細表示仕様（別仕様書）
- 再生制御機能の動作仕様（別仕様書）
- レビュー投稿セクションの詳細（別仕様書）

## 基本情報セクション

### 概要

再生リストの基本的な情報（タイトル、評価、ユーザー数、タグ、参照元URL）をコンパクトに表示するセクション。ユーザーが「このシリーズの信頼度」「どの程度のユーザーに視聴されているか」を素早く判断できるよう設計。

### 表示順序（上から下へ）

1. 再生リストのタイトル
2. スコア、マイリスト数、レビュー数
3. 付与されたタグ
4. 参照元URL（YouTubeの元の再生リストへのハイパーリンク）

### 各項目の仕様

| 項目 | 詳細 |
|------|------|
| 再生リストのタイトル | 公式タイトル、最大1行表示、超過時は省略記号（…）で切り捨て |
| スコア | 信頼度加重平均によるシリーズスコア。表示仕様は信頼度スコアリングシステム仕様書を参照 |
| マイリスト数 | このシリーズをマイリストに登録したユーザー数（数値表示） |
| レビュー数 | 投稿されたレビューの総数（数値表示） |
| タグ | この再生リストに付与されたタグ（バッジ形式で複数表示） |
| 参照元URL | YouTubeの元の再生リストへのハイパーリンク（新しいタブで開く） |

### 配置イメージ

```
┌─────────────────────────────────┐
│ 再生リストのタイトル             │
│ （最大1行、超過時は…）           │
│                                 │
│ ★★★★★ 4.5 / 5                  │
│ マイリスト: 234 | レビュー: 89   │
│                                 │
│ [ジャンル1] [ジャンル2]          │
│ [テーマ1]                        │
│                                 │
│ YouTubeの再生リストを見る ↗      │
└─────────────────────────────────┘
```

## 配信者情報セクション

### 概要

再生リストを投稿した配信者（YouTubeチャンネル）の基本情報を表示するセクション。ユーザーが同じ配信者の他の実況シリーズへ自然に導線をたどれるよう設計。

### 表示位置

PC版：右カラムの、動画リストの下、ゲーム情報セクションの上

モバイル版：レビュー投稿セクションの下、ゲーム情報セクションの上

### 表示レイアウト

```
┌────────────────────────────────┐
│  ⚬ チャンネルアイコン（円形）    │
│                                │
│  チャンネル名                   │
│                                │
│  [他の再生リストを見る →]        │
└────────────────────────────────┘
```

### 表示要素の仕様

| 項目 | 詳細 |
|------|------|
| チャンネルアイコン | YouTubeから取得した円形のチャンネルアイコン画像 |
| チャンネル名 | 実況者のYouTubeチャンネル名 |
| 他の再生リストを見るリンク | チャンネル詳細ページへの内部リンク |

### クリック動作

| 要素 | クリック動作 |
|------|------------|
| チャンネルアイコン | チャンネル詳細ページ（`/channels/[channelId]`）へ遷移 |
| チャンネル名 | チャンネル詳細ページ（`/channels/[channelId]`）へ遷移 |
| 他の再生リストを見るリンク | チャンネル詳細ページ（`/channels/[channelId]`）へ遷移 |

## ゲーム情報セクション

### 概要

再生リストに紐づくゲームタイトルの基本情報をコンパクトに表示するセクション。右カラムの幅に合わせた**1カラム縦積み構成**とする。楽天ブックスへのアフィリエイト導線と、ゲームタイトル詳細ページへの導線を設ける。

### 表示位置

PC版：右カラムの、配信者情報セクションの下

モバイル版：配信者情報セクションの下（ページ最後、フッターの前）

### 表示レイアウト

```
┌────────────────────────────────┐
│  ゲームタイトル名               │
│                                │
│  ┌──────────────────────┐     │
│  │    パッケージ画像     │     │
│  │      （縦長）        │     │
│  └──────────────────────┘     │
│  [楽天ブックスで見る ↗]        │
│                                │
│  ジャンル：RPG / アクション     │
│  テーマ：冒険 / ダンジョン      │
│                                │
│  [タグ] [タグ] [タグ]          │
│                                │
│  [このゲームの再生リストを       │
│         見る →]               │
└────────────────────────────────┘
```

### 表示要素の仕様

| 項目 | 詳細 |
|------|------|
| ゲームタイトル名 | DBに登録されたゲームタイトル名。最大2行、超過時は省略記号（…）で切り捨て |
| パッケージ画像 | 楽天ブックスAPIから取得したゲームパッケージ画像（アスペクト比 約3:4、縦長）。右カラム幅いっぱいに表示。未取得の場合はプレースホルダー画像を表示 |
| 楽天ブックスで見るボタン | パッケージ画像の直下に配置。楽天ブックス商品詳細ページへのアフィリエイトリンク（別タブ） |
| ジャンル | DBのジャンルマスタから取得。複数の場合は「/」区切りで表示（例：「RPG / アクション」） |
| テーマ | DBのテーママスタから取得。複数の場合は「/」区切りで表示（例：「冒険 / ダンジョン」） |
| タグ | このゲームタイトルに付与されたタグをバッジ形式で表示。表示優先順位は編集不可タグ優先 |
| このゲームの再生リストを見るリンク | ゲームタイトル詳細ページへの内部リンク |

### クリック動作

| 要素 | クリック動作 |
|------|------------|
| パッケージ画像 | 楽天ブックス商品詳細ページ（アフィリエイトリンク）へ別タブで遷移 |
| 楽天ブックスで見るボタン | 楽天ブックス商品詳細ページ（アフィリエイトリンク）へ別タブで遷移 |
| タグ | 「ゲームタイトルから探す」ページへ遷移し、クリックしたタグが検索条件として適用された状態で表示する |
| このゲームの再生リストを見るリンク | ゲームタイトル詳細ページ（`/games/[gameId]`）へ遷移 |

### 楽天ブックスアフィリエイトリンク

| 項目 | 仕様 |
|------|------|
| ボタン文言 | 「楽天ブックスで見る ↗」 |
| 遷移先 | 楽天ブックス商品詳細ページ（アフィリエイトリンク） |
| 開き方 | 別タブ（`target="_blank"`） |
| アフィリエイトキー | 別途定義 |
