## 1. 概要

利用規約ページは、ゲムコメの利用規約を掲載するページである。

利用規約の本文は後日整備する。本仕様書では、ページの存在とアクセス導線を定義することを目的とする。

---

## 2. ページURL・アクセス条件

| 項目 | 内容 |
|------|------|
| URL | `/terms` |
| アクセス条件 | ログイン・未ログイン問わず全ユーザーがアクセス可能 |
| ヘッダーナビゲーション | 常に表示 |

---

## 3. 流入導線

### 3.1 フッターからの導線

全ページ共通のフッターにリンクを常設する。

| 項目 | 内容 |
|------|------|
| 表示条件 | ログイン・未ログイン問わず常時表示 |
| リンク文言 | 「利用規約」 |
| 配置位置 | フッター内のリンク群に配置 |

### 3.2 アカウント登録フローからの導線

アカウント作成フォームに、利用規約への同意文を設置する。

| 項目 | 内容 |
|------|------|
| 配置位置 | 「アカウントを作成する」ボタンの直上 |
| 表示文言 | 「アカウントを作成することで、[利用規約]に同意したものとみなします。」 |
| リンク | 「利用規約」部分をリンクとし、クリックで利用規約ページへ遷移する |
| 遷移方式 | 別タブで開く（`target="_blank"`） |

> ソーシャルログイン（Google・X）経由でのアカウント作成にも同様の同意文を表示する。各プロバイダのソーシャルログインボタンの直上に配置する。
