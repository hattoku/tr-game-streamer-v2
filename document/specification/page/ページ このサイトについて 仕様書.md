## 1. 概要

「このサイトについて」ページは、プレミテの運営者情報・サービス概要・免責事項を掲載するページである。サービスの透明性を確保し、ユーザーが運営主体を確認できる場として機能する。

---

## 2. ページURL・アクセス条件

| 項目 | 内容 |
|------|------|
| URL | `/site-info` |
| アクセス条件 | ログイン・未ログイン問わず全ユーザーがアクセス可能 |
| ヘッダーナビゲーション | 常に表示 |

---

## 3. 流入導線

### 3.1 フッターからの導線

全ページ共通のフッターにリンクを常設する。詳細は「共通 UIコンポーネント フロント 仕様書」を参照。

| 項目 | 内容 |
|------|------|
| 表示条件 | ログイン・未ログイン問わず常時表示 |
| リンク文言 | 「このサイトについて」 |
| 配置位置 | フッター内のサポートグループ |

---

## 4. ページ構成

セクションは上から順に以下のとおり配置する。

```
ヘッダー
  ↓
① サービス概要・運営者情報セクション
  ↓
② 免責事項セクション
  ↓
③ お問い合わせ導線セクション
  ↓
フッター
```

---

## 5. 各セクションの仕様

### 5.1 サービス概要・運営者情報セクション

サービスの内容と運営主体を記載する。

| 項目 | 内容 |
|------|------|
| サービス名 | プレミテ |
| サービス概要 | ゲーム実況動画の再生リスト視聴記録・新着通知サービス |
| 運営者名 | 発見&納得コミュニケーションズ |

> 住所・代表者名・連絡先等の詳細情報は後日整備する。

### 5.2 免責事項セクション

免責事項の本文を掲載する。本文は後日整備する。本仕様書では、セクションの存在とページ内での配置を定義することを目的とする。

| 要素 | 内容 |
|------|------|
| セクション見出し | 「免責事項」 |
| 本文エリア | 免責事項の本文を掲載する（本文は後日整備） |

### 5.3 お問い合わせ導線セクション

お問い合わせページへの導線を設置する。

| 項目 | 内容 |
|------|------|
| セクション見出し | 「お問い合わせ」 |
| 本文 | 「ご不明な点やお気づきの点がございましたら、お問い合わせフォームよりご連絡ください。」 |
| リンク文言 | 「お問い合わせフォームへ」 |
| 遷移先 | `/contact` |
