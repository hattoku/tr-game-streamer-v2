## 1. 概要

### 1.1 機能概要

「チャンネル詳細ページ」は、「チャンネル名から探す」ページでユーザーが選択したチャンネルの詳細情報と、そのチャンネルに紐づく再生リスト一覧を掲載するページである。

ユーザーはこのページを通じて、特定のチャンネルの基本情報（サムネイル・再生リスト数・総動画数・説明文）を確認しつつ、そのチャンネルが持つ再生リストへ到達できる。また、未登録再生リストの登録を促す導線も設けることで、コンテンツ充実を図る。

### 1.2 アクセス条件

| 条件 | 内容 |
|------|------|
| ログイン済みユーザー | ✅ アクセス可能 |
| 未ログインユーザー | ✅ アクセス可能（閲覧は可能。再生リスト追加はログイン必須） |

### 1.3 ページURL

| 項目 | 内容 |
|------|------|
| URL | `/channels/[channelId]` |

## 3. チャンネル情報セクション

### 3.1 レイアウト方針

PCとスマートフォンでレイアウト構造が異なる。

| デバイス | 構造 |
|---------|------|
| PC | サムネイル（左）とチャンネル名・統計情報・ボタン（右）を横並び表示。説明文はその下に全幅で表示 |
| スマートフォン | サムネイルを上、テキスト情報（チャンネル名・統計情報・ボタン）を下に縦積み表示。説明文はその下に表示 |

### 3.2 PC表示レイアウト

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│  ┌──────────────┐  チャンネル名                                  │
│  │              │                                               │
│  │  チャンネル  │  □ 3 再生リスト  ▶ 45 動画                   │
│  │  サムネイル  │                                               │
│  │  （正方形）  │  [再生リストを追加する]                        │
│  │              │                                               │
│  └──────────────┘                                               │
│                                                                 │
│  チャンネルの説明（設定済みの場合のみ表示）                       │
│  ┌─────────────────────────────────────────────────────────┐    │
│  │ チャンネルの紹介文テキスト                               │    │
│  │ [AIによる生成] バッジ（フラグONの場合のみ表示）          │    │
│  │                              [編集する]（管理者（オーナー・運営者）のみ）    │    │
│  └─────────────────────────────────────────────────────────┘    │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 3.3 表示要素の仕様

#### チャンネルサムネイル

| 項目 | 仕様 |
|------|------|
| 取得元 | YouTubeチャンネルのアイコン画像 |
| アスペクト比 | 正方形 |
| 未取得時 | デフォルトのプレースホルダー画像を表示 |
| クリック動作 | なし |

#### チャンネル名

| 項目 | 仕様 |
|------|------|
| 表示内容 | YouTubeチャンネル名（DB登録値） |
| 最大行数 | 制限なし（改行して全文表示） |

#### 再生リスト数・総動画数

| 項目 | 仕様 |
|------|------|
| 表示位置 | チャンネル名の下、ボタンの上 |
| 再生リスト数の表示形式 | 再生リストを示すアイコン +「再生リスト」テキスト + 数値（例：□ 3 再生リスト） |
| 総動画数の表示形式 | 動画再生を示すアイコン +「動画」テキスト + 数値（例：▶ 45 動画） |
| 再生リスト数の定義 | このチャンネルに紐づく再生リストの登録数合計 |
| 総動画数の定義 | このチャンネルに紐づく全再生リストの動画数合計 |
| アイコン | 実装者に委ねる（「チャンネル名から探す」のカード表示と統一すること） |

#### 再生リストを追加するボタン

ログイン済みのユーザー種別によってボタンの動作が変わる。遷移先は共通だが、「再生リストを追加する」ページ側でユーザー種別を判定し、管理者（オーナー・運営者）は登録フロー・一般ユーザーは提案フローへ分岐する（「再生リストを追加する」仕様書 第1章・第5章参照）。

| ユーザー種別 | 表示・動作 |
|------------|----------|
| 未ログイン | 「再生リストを追加する」ボタンを表示。クリック時はログインを促すダイアログを表示、またはログインページへ遷移 |
| 一般ユーザー | 「再生リストの登録を提案する」ボタンを表示。クリック時は「再生リストを追加する」ページへ遷移し、チャンネルが固定指定された状態で開く（提案フロー） |
| 管理者（オーナー・運営者） | 「再生リストを追加する」ボタンを表示。クリック時は「再生リストを追加する」ページへ遷移し、チャンネルが固定指定された状態で開く（登録フロー） |

#### チャンネルの説明文

| 項目 | 仕様 |
|------|------|
| 表示条件 | DBに説明文が登録されている場合のみ表示。未設定の場合は説明文欄自体を表示しない |
| 表示位置 | サムネイル・チャンネル名・統計情報・ボタンの下、全幅で表示 |
| 最大行数 | 制限なし（全文表示） |

#### AIによる生成バッジ

| 項目 | 仕様 |
|------|------|
| 表示条件 | 説明文が表示されており、かつAI生成バッジフラグがONの場合のみ表示 |
| 表示内容 | 「AIによる生成」のバッジ |
| 表示位置 | 説明文テキストに隣接して表示（実装者に委ねる） |

#### 説明文の編集（管理者（オーナー・運営者）のみ）

| 項目 | 仕様 |
|------|------|
| 表示条件 | 管理者（オーナー・運営者）でログイン中、かつ説明文が表示されている場合のみ「編集する」ボタンを表示 |
| クリック時の動作 | 説明文テキストをテキストエリアに切り替えてインライン編集モードに移行する |
| 編集モードの構成 | テキストエリア＋「保存する」ボタン＋「キャンセル」ボタン＋AI生成バッジ表示トグル |
| 「保存する」クリック時 | 編集内容をDBに保存し、表示モードに戻る |
| 「キャンセル」クリック時 | 編集内容を破棄し、表示モードに戻る |
| AI生成バッジ表示トグル | ON/OFFを切り替えることでAI生成バッジフラグをDBに保存する。動作仕様は「再生リストを追加する」仕様書（4.1.4節）に準ずる |

## 4. 再生リスト一覧セクション

### 4.1 セクション構成

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│  XX件  ソート: [スコアが高い順 ▼]                           │
│                                                             │
│  ┌───────────────────────────────────────────────────────┐  │
│  │ カード                                                │  │
│  └───────────────────────────────────────────────────────┘  │
│  ┌───────────────────────────────────────────────────────┐  │
│  │ カード                                                │  │
│  └───────────────────────────────────────────────────────┘  │
│  ...                                                        │
│                                                             │
│  < 1  2  3 >                                                │
└─────────────────────────────────────────────────────────────┘
```

### 4.2 カードの仕様

カードのデザインおよび表示要素は「再生リストを探す」仕様書（第3章）で定義したカードと同一とする。

### 4.3 一覧ヘッダー

| 表示要素 | 詳細 |
|---------|------|
| 検索結果件数 | 「XX件」の形式で表示 |
| ソートUI | 件数表示の右側にドロップダウンを設置 |

### 4.4 ソート仕様

| ソート | 説明 | デフォルト |
|--------|------|-----------|
| スコアが高い順 | 加重平均スコアの高い順に表示 | ✅ デフォルト |
| マイリスト登録者数順 | マイリストへの登録者数が多い順に表示 | — |
| 新着順 | 再生リスト内の最新動画の投稿日が新しい順に表示 | — |

> **設計思想**: デフォルトを「スコアが高い順」とすることで、初めてページを訪れたユーザーが「評判の良い再生リスト」を最初に目にできる導線を設計する。ソートの詳細仕様（新着順の定義・スコアなしの扱い等）は「再生リストを探す」仕様書（第5章）に準ずる。

### 4.5 ソートUIの仕様

| 項目 | 仕様 |
|------|------|
| UI形式 | ドロップダウン（セレクトボックス） |
| 配置位置 | 一覧ヘッダーの件数表示の右側 |
| 表示文言 | 選択中のソート名を表示（例：「スコアが高い順」） |

### 4.6 ページネーション

| 項目 | 仕様 |
|------|------|
| 1ページあたりの表示件数 | 20件 |
| 方式 | ページネーション（ページ番号リンク形式） |
| 表示位置 | カード一覧の下部 |

## 5. 空状態の表示

このチャンネルに紐づく再生リストが0件の場合（まだゲムコメにこのチャンネルの再生リストが登録されていない場合）は以下を表示する。

| 要素 | 内容 |
|------|------|
| アイコン | 再生リスト・動画を示すアイコン（実装者に委ねる） |
| メッセージ | 「まだ登録されている再生リストがありません」 |
| サブメッセージ | 「最初の再生リストを登録してみませんか？」 |
| ボタン | 管理者（オーナー・運営者）・未ログインは「再生リストを追加する」、一般ユーザーは「再生リストの登録を提案する」を表示（権限制御は [3.3節](#33-表示要素の仕様) に準ずる） |
