## 1. デザイン方針

### 1.1 コンセプト

プレミテのUIは「シネマダーク」をベースとする。動画視聴が主体のサービスとして、サムネイルやコンテンツが主役になるよう、UIクローム（ナビ・ボタン・カード）は背景に溶け込む設計とする。

参考UIとして YouTube シアターモード・Amazon Prime Video を参照している。

### 1.2 カラー使用の原則

| 色 | 用途 |
|----|------|
| 白・グレー系 | UI全般（背景・ボタン・テキスト） |
| 赤（`#e03030`） | ロゴアイコンのみ. それ以外には使用しない |
| アンバー（`#f0a500`） | ★スコア表示のみ. それ以外には使用しない |
| シアングリーン（`#1db37a`） | 「視聴中」ステータスのアクティブ表示のみ |
| スカイブルー（`#38a8f5`） | 「注目上昇中」バッジのみ |
| オレンジレッド（`#bf360c`） | 楽天ブックスボタンのみ |

刺し色は用途を厳格に限定し、多用しない。

## 2. カラートークン

### 2.1 背景

| トークン名 | 値 | 用途 |
|-----------|-----|------|
| `color-bg-base` | `#0f0f0f` | ページ背景 |
| `color-bg-card` | `#141414` | カード・パネル背景 |
| `color-bg-btn` | `#252525` | セカンダリボタン・インタラクティブ要素背景 |
| `color-bg-input` | `#1a1a1a` | ステータスチップ・入力要素背景 |
| `color-bg-player` | `#000000` | 動画プレーヤー背景 |

### 2.2 ボーダー

| トークン名 | 値 | 用途 |
|-----------|-----|------|
| `color-border-subtle` | `#1e1e1e` | カード内セパレーター |
| `color-border-default` | `#222222` | カード外枠 |
| `color-border-strong` | `#3a3a3a` | ボタン枠・強調ボーダー |
| `color-border-chip` | `#2e2e2e` | ステータスチップ（非アクティブ） |

### 2.3 テキスト

テキストは5段階の明度で役割を分ける。

| トークン名 | 値 | 用途例 |
|-----------|-----|--------|
| `color-text-primary` | `#e8e8e8` | メインタイトル・再生中動画タイトル |
| `color-text-secondary` | `#cccccc` | 重要な補足情報（話数・年・ジャンル値） |
| `color-text-tertiary` | `#aaaaaa` | タグ・サブ情報・非アクティブチップ |
| `color-text-muted` | `#888888` | ラベル（「ジャンル：」など）・メタ情報 |
| `color-text-faint` | `#777777` | 動画尺・エピソード番号など最小情報 |

### 2.4 ブランド・アクセントカラー

| トークン名 | 値 | 用途 |
|-----------|-----|------|
| `color-brand-logo` | `#e03030` | ロゴアイコン背景のみ |
| `color-brand-score` | `#f0a500` | ★スコア数値・星アイコン |
| `color-brand-watching` | `#1db37a` | 「視聴中」ステータスアクティブ色 |
| `color-brand-watching-bg` | `#0d2e2a` | 「視聴中」チップ背景 |
| `color-brand-trending` | `#38a8f5` | 「注目上昇中」バッジ色 |
| `color-brand-trending-bg` | `#0a1e2e` | 「注目上昇中」バッジ背景 |

| `color-progress-bar` | `#e03030` | 動画視聴進捗バー（YouTube準拠） |

## 3. タイポグラフィ

### 3.1 フォント

| 項目 | 値 |
|------|----|
| フォントファミリー | システムフォント（`-apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif`） |
| ベースフォントサイズ | `14px` |
| ベース行間 | `1.6` |

### 3.2 フォントサイズ一覧

| トークン名 | 値 | 用途例 |
|-----------|-----|--------|
| `font-size-xs` | `10px` | エピソード番号・進捗バー補足 |
| `font-size-sm` | `11px` | 動画尺・ゲームタグ・最小ラベル |
| `font-size-md` | `12px` | 動画タイトル・カードメタ情報・チップ |
| `font-size-base` | `13px` | ナビリンク・ボタンテキスト・本文 |
| `font-size-lg` | `14px` | チャンネル名・ゲームタイトル |
| `font-size-xl` | `15px` | ロゴテキスト・セクションタイトル |
| `font-size-2xl` | `16px` | 再生リストタイトル |
| `font-size-score` | `26px` | シリーズスコア数値 |

### 3.3 フォントウェイト

| トークン名 | 値 | 用途 |
|-----------|-----|------|
| `font-weight-normal` | `400` | 本文・メタ情報 |
| `font-weight-medium` | `500` | タイトル・ボタン・見出し |

## 4. ボタン

ボタンは3段階の優先度で使い分ける。

### 4.1 プライマリボタン（最重要アクション）

「視聴する」「続きから再生」など、ページ内で最も重要な1アクションに使用する。1ページにつき原則1つ。

| 項目 | 値 |
|------|----|
| 背景色 | `#e8e8e8` |
| テキスト色 | `#111111` |
| フォントサイズ | `14px` |
| フォントウェイト | `500` |
| 角丸 | `7px` |
| パディング | `9px 22px` |

```css
background: #e8e8e8;
color: #111111;
font-size: 14px;
font-weight: 500;
border-radius: 7px;
padding: 9px 22px;
border: none;
```

### 4.2 セカンダリボタン（重要アクション）

「マイリストに追加」「前へ」「次へ」「シアター」「連続再生」など操作系に使用する。

| 項目 | 値 |
|------|----|
| 背景色 | `#252525` |
| テキスト色 | `#e0e0e0` |
| ボーダー | `0.5px solid #3a3a3a` |
| フォントサイズ | `13px` |
| フォントウェイト | `500` |
| 角丸 | `7px` |
| パディング | `8px 14px`（操作系） / `10px`（幅100%のアクション系） |

```css
background: #252525;
color: #e0e0e0;
font-size: 13px;
font-weight: 500;
border-radius: 7px;
padding: 8px 14px;
border: 0.5px solid #3a3a3a;
```

### 4.3 外部リンクボタン（サードパーティ）

楽天ブックスなど、外部サービスへ誘導するボタン。各サービスのブランドカラーを使用する。

| サービス | 背景色 | テキスト色 |
|---------|--------|-----------|
| 楽天ブックス | `#bf360c` | `#ffffff` |

```css
background: #bf360c;
color: #ffffff;
font-size: 12px;
font-weight: 500;
border-radius: 6px;
padding: 6px 12px;
border: none;
```

## 5. バッジ・ステータス表示

### 5.1 視聴ステータスチップ

レビューセクションの視聴ステータス選択UIに使用する。

#### 非アクティブ状態

```css
background: #1a1a1a;
color: #aaaaaa;
border: 0.5px solid #2e2e2e;
border-radius: 20px;
font-size: 12px;
padding: 5px 12px;
```

#### アクティブ状態（「視聴中」選択時）

「視聴中」選択時のみシアングリーンを使用する。他のステータス（完走・見たい・一時中断・断念）のアクティブ色は別途定義する。

```css
background: #0d2e2a;
color: #1db37a;
border: 0.5px solid #1db37a;
border-radius: 20px;
font-size: 12px;
padding: 5px 12px;
/* 左端にドットを付与 */
```

ドット仕様：
- サイズ：`5px × 5px`
- 形状：円形（`border-radius: 50%`）
- 色：`#1db37a`
- テキストとの間隔：`5px`

### 5.2 注目上昇中バッジ

再生リスト基本情報カードの右上に表示する。表示条件は別途ロジックで定義する。

```css
background: #0a1e2e;
color: #38a8f5;
border: 0.5px solid #38a8f5;
border-radius: 4px;
font-size: 11px;
font-weight: 500;
padding: 3px 8px;
```

表示内容：`↑ 注目上昇中`

## 6. カード・コンテナ

### 6.1 標準カード

右カラムの各セクション（基本情報・チャンネル情報・ゲーム情報など）に使用する。

```css
background: #141414;
border: 0.5px solid #222222;
border-radius: 10px;
padding: 18px;
```

### 6.2 カード内セパレーター

```css
border-top: 0.5px solid #1e1e1e;
```

### 6.3 動画リストアイテム（再生中ハイライト）

```css
background: #1c1c1c;
```

### 6.4 視聴進捗バー

動画リスト内のサムネイル下に表示する細い横線。

| 項目 | 値 |
|------|----|
| 高さ | `2px` |
| 未視聴部分 | `#333333` |
| 視聴済み部分 | `#e03030`（YouTube準拠） |
| 角丸 | `2px` |
| 配置 | サムネイル直下（`margin-top: 2px`） |

## 7. タグ

クリックで検索・絞り込みにつながるテキストリンク形式のタグ。ボタン型は使用しない。

```css
color: #aaaaaa;
font-size: 12px;  /* 再生リストタグ */
text-decoration: underline;
text-decoration-color: #555555;
text-underline-offset: 3px;
background: none;
border: none;
cursor: pointer;
```

ゲーム情報内のタグは `font-size: 11px` とする。

## 8. アイコンリンク

### 8.1 YouTubeチャンネルリンク

チャンネル情報カード内に配置する。YouTubeブランドアイコン＋白テキストの組み合わせ。

| 項目 | 値 |
|------|----|
| アイコン | YouTubeロゴ SVG（赤矩形 `#FF0000` ＋ 白三角） |
| アイコンサイズ | `16px × 11px` |
| テキスト | `で見る ↗` |
| テキスト色 | `#e0e0e0` |
| フォントサイズ | `12px` |
| アイコンとテキストの間隔 | `5px` |

### 8.2 ロゴアイコン

ヘッダーのサービスロゴアイコン部分。将来的には画像ロゴに置き換える。

| 項目 | 値 |
|------|----|
| 背景色 | `#e03030` |
| サイズ | `26px × 26px` |
| 角丸 | `6px` |
| アイコン内容 | 再生三角形（白・SVG） |
| テキスト「プレミテ」色 | `#e8e8e8` |
| テキストサイズ | `15px` |
| テキストウェイト | `500` |
