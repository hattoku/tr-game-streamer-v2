## 2. 競合サービス分析

### 2.1 Metacritic（ゲームソフト評価プラットフォーム）

#### 概要
- プロ批評家レビュー（Metascore）とユーザーレビュー（User Score）の二層構造
- 100点満点のスコアリングシステム
- 業界標準の評価プラットフォーム

#### 評価システムの特徴

| 要素 | Metascore（批評家） | User Score（ユーザー） |
|------|-------------------|----------------------|
| 評価者 | 認定メディア約100社 | 一般ユーザー |
| スコア表示 | 0-100点 | 0-10点 |
| 計算方法 | 加重平均 | 単純平均（公式）<br>実際は重み付けあり（推定） |
| データ収集 | 手動キュレーション | ユーザーが直接投稿 |

#### Metascoreの仕組み
```
1. 認定メディアがレビューを公開
   ↓
2. Metacriticの編集チームが発見・収集
   ↓
3. 各メディアの評価形式を100点満点に正規化
   例：IGN 9.0/10 → 90/100
       GameSpot 4/5 → 80/100
       Polygon "Buy" → 90/100（推定変換）
   ↓
4. メディアの影響力による加重平均を計算
   ↓
5. Metascoreとして表示
```

#### User Scoreの重み付け（推定）
```
公式説明：単純平均
実態：以下の要素で重み付けされている（推定）

重み付け要素：
- アカウント信頼度（新規 vs 長期ユーザー）
- レビューの質（文字数、詳細度）
- 投稿タイミング（Review Bombing対策）
- 極端なスコアへのペナルティ
- コミュニティ評価（「役立った」投票）
